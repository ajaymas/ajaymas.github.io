About me

Welcome to my home page!

I am an Assistant Professor of Computer Science at Lenior-Rhyne University. My research interests focus on Cybersecurity, Binary Code Analysis, Malware Analysis, Reverse Engineering, Virtualization, and Deep Learning.

Before joining Lenior-Rhyne, I worked as a Postdoctoral Researcher at the Pennsylvania State University at the system software security lab and the University of Arizona. I completed my Ph.D., from National Institute of Technology, Karnataka, India.

Research Interest

My primary research interests span broadly across the areas as follows:
  • 1. Static and Dynamic Binary Code Analysis
  • 2. Virtual Machine Introspection and Memory Forensics
  • 3. Cloud Computing and Virtualization
  • 4. Software Reverse Rngineering and Software Security
  • 5. Behavioural Analysis of Malware
  • 6. Machine Learning and Deep Learning for Security
